# Data Directory: Oslofjord_nutrients

This directory contains the raw datasets used to analyze the multi-decadal physical, chemical, and biological trajectories of the Oslofjord ecosystem (1930â€“2025). 

## Directory Structure & File Manifest
### `/raw/`
This folder contains the unmodified, original datasets aggregated from external repositories. 
* **`WaterRegistrationExport_f[1-3].xlsx`**: Standardized physical and biogeochemical monitoring data extracted from the Norwegian Environment Agency's VannmiljĂ¸ database. 
* **`event.txt`**: Marine observations extracted from the GBIF Archive. Critical for establishing the century-scale baseline (pre-eutrophication and peak-eutrophication eras).
* **`measurementorfacts.txt`**: Marine observations extracted from the GBIF Archive. Critical for establishing the century-scale baseline (pre-eutrophication and peak-eutrophication eras).

* **`satellite_modis.csv`**: Daily surface Chlorophyll-*a* concentrations acquired from the Copernicus Marine Environment Monitoring Service (CMEMS), specifically the `erdMH1chla8day` regional MODIS ocean color product (2003â€“2023).

### `/processed/`
This folder contains the intermediate data objects generated by the `00_data_integration_QAQC.R` script. These `.rds` (R Data) files are the direct inputs for all downstream GAMs, PCAs, and ggplot visualizations.

* **`oslo_final_sf.rds`**: The master in situ spatial dataframe (an `sf` object). This dataset has already undergone rigorous QA/QC, including:
  * Harmonization of variable nomenclature via regular expressions.
  * Spatial aggregation into 104 distinct 1km-radius site clusters.
  * Hierarchical Interquartile Range (IQR) outlier filtering (grouped by region, depth, and season).
  * Bias-free averaging to prevent historical summer-sampling bias from skewing annual means.
* **`satellite_cleaned.rds`**: The processed remote sensing dataframe. The raw satellite data has been spatially masked to match the exact boundaries of the four fjord regions, with strict cloud-masking and coastal flags applied to prevent optical contamination from landmasses.

## Key Data Dictionary

When loading `oslo_final_sf.rds`, the following key variables are utilized across the analytical scripts:

| Variable | Type | Description |
| :--- | :--- | :--- |
| `date` | Date | Date of the sampling event (YYYY-MM-DD). |
| `Fjord_Part` | Factor | The hydrographic region: `Drammensfjord`, `Inner Oslofjord`, `Outer East`, or `Outer West`. |
| `Depth_Bin` | Factor | Categorized vertical horizon. Primary analyses utilize `0-5m (Surface)` and `5-20m (Pycnocline/DCM)`. |
| `parameter` | Character | The measured environmental variable (e.g., `Chla`, `Temperature`, `Salinity`, `Secchi`, `Nitrate`, `Phosphate`, `Total_Nitrogen`, `Total_Phosphorus`). |
| `value` | Numeric | The raw concentration or measurement of the parameter. |
| `geometry` | sfc_POINT | The exact spatial coordinates of the 1km cluster. |

*Note: Stoichiometric molar ratios (e.g., NO3:PO4) and seasonally adjusted parameters are calculated dynamically within the respective module scripts (e.g., `scripts/03_multivariate_PCA_RDA.R`) rather than hard-coded into the processed dataframe.*
